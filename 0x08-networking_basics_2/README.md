# 0x08. Networking Basics #1

This folder contains projects and assignments related to the topic of Networking Basics #1.

## Description

This project is part of the curriculum at [Holberton School](https://www.holbertonschool.com/), covering the fundamentals of computer networking. In this repository, you'll find various tasks and assignments related to networking basics, including topics such as IP addresses, localhosts, DNS resolution, TCP/IP, and more.

## Contents

- **0-OSI_model**: Introduction to the OSI model and its layers.
- **1-types_of_network**: Explanation of LAN, WAN, and the Internet.
- **2-MAC_and_IP_addressing**: Overview of MAC and IP addresses, along with the ARP protocol.
- **3-UDP_and_TCP**: Introduction to the UDP and TCP protocols.
- **4-TCP_and_UDP_ports**: Explanation of TCP and UDP ports.
- **5-is_the_host_on_the_network**: Bash script to ping an IP address passed as an argument.
- **6-what_smy_IP**: Bash script to display the IP addresses of the computer.

## Usage

Each directory contains scripts or explanations related to specific networking topics. You can navigate to each directory to find more detailed instructions on how to use or understand the content.

To clone this repository, use the following command:

```bash
git clone https://github.com/lesuuda/alx-system_engineering-devops/0x08-networking-basics-1.git
```
